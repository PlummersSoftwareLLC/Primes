Primes based on the code from Dave Plummer Primes repository.

A Rustlang copy of the algorithm presented in PrimeCPP.

## HOW TO BUILD / RUN

Install Rust

then:

```
    .\run.cmd (Windows)
    or
    ./run.sh (*nix)
```

with docker:

```
    docker build . -t primerust
    docker run primerust
```
